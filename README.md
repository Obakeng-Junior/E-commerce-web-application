**Team Project | .NET, MVC, SQLite, REST APIs**

Dark Hawk Security is an e-commerce platform for security services and products. Customers can book services, browse products, add items to their cart, and checkout. Admins manage bookings, complaints, product stock, and ensure smooth operations.

---

## Features

### Customer Side
- **Service Booking:** Customers can book security services.
- **Product Catalog:** Browse available products, add to cart, and checkout.
- **Complaints & Support:** Submit complaints or inquiries and track admin responses.
- **Order Tracking:** View the status of service bookings and product orders.
- **Secure Authentication:** User accounts are protected and sensitive data is secured.

### Admin Side
- **Dashboard:** View and manage all bookings, complaints, product catalog, and stock levels.
- **Complaint Management:** Respond to customer inquiries.
- **Product Management:** Add, update, and manage stock of products.
- **Data Management:** Optimize database operations and maintain integrity.
- **Secure Workflows:** REST APIs ensure safe and reliable operations.

---

## Technologies Used
- **Backend:** ASP.NET MVC, C#
- **Database:** SQLite
- **API Integration:** REST APIs for secure communication
- **Version Control:** Git & GitHub

---

## How It Works

1. **User Registration & Login:** Customers create accounts to access booking and shopping features.
2. **Booking Services:** Select and book security services.
3. **Shopping:** Browse products, add to cart, and checkout.
4. **Complaints:** Submit inquiries or complaints via the system.
5. **Admin Management:** Admin manages stock, products, bookings, and responds to complaints.
6. **Tracking:** Customers track bookings, product orders, and responses.

---

## Project Structure
- **Controllers:** Application logic for users and admins.
- **Models:** Data structures for Users, Products, Bookings, and Complaints.
- **Views:** Razor pages for front-end UI.
- **Database:** SQLite stores users, products, stock, bookings, and complaints.
- **API Endpoints:** Secure communication between front-end and backend.
